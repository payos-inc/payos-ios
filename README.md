# PayOS iOS SDK

Native iOS SDK for PayOS.

## Installation

Add the package with Swift Package Manager:

```swift
.package(url: "https://github.com/payos-inc/payos-ios.git", from: "0.1.0-staging.4.2")
```

Then import the SDK:

```swift
import PayOS
```

Access to this private package requires GitHub read access to `payos-inc/payos-ios`.

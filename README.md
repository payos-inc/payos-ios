# PayOS iOS SDK

Native iOS SDK for PayOS.

## Installation

Add the package with Swift Package Manager:

```swift
.package(url: "https://github.com/payos-inc/payos-ios.git", from: "0.1.0")
```

Then import the SDK:

```swift
import PayOS
```
